# Patience Konde Portfolio Website

A personal portfolio website built with HTML and CSS, showcasing a professional introduction, about section, projects, portfolio gallery, and contact form.

## Overview

This site includes:
- A header section with profile image and introduction
- An About section with tabs for skills, experience, and education
- A Projects section displaying sample projects with call-to-action links
- A Portfolio gallery with hover interaction
- A Contact section with email, phone, social icons, and a contact form powered by Formspree

## Technologies

- HTML
- CSS
- Font Awesome icons
- Formspree for contact submissions

## Files

- `index.html` — main page structure
- `styles.css` — page styling and layout
- `images/` — profile and portfolio images

## Usage

1. Open `index.html` in a browser.
2. Optionally host the project on GitHub Pages or another static site host.

## Notes

- Update the contact email, phone number, and portfolio links as needed.
- Replace placeholder project links with your live demo URLs.
- Ensure the `images` folder contains the referenced image files.
